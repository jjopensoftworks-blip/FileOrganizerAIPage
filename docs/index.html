Truly Offline

No account, no telemetry, no uploads. The only optional network call is a one-time AI model download you have to opt into — decline it and the app never touches the network.

bolt

Instant by Default

A fast local classifier sorts by extension and filename rules — invoices, screenshots, receipts, installers — in seconds, with no model to download.

Optional On-Device AI

Turn on a local Phi-3.5 model (via ONNX Runtime + DirectML) for smarter categories on tricky files. It runs entirely on your GPU — nothing is sent anywhere.

Nothing moves until you say so. Review and edit every proposed folder, choose Move or Copy, and reverse the whole run with one click.

How it works

STEP 1

folder_open

Pick a folder

Choose any messy folder — Downloads, Desktop, a scanner dump. It's read first; nothing changes.

STEP 2

rule

It proposes

A 3-layer classifier suggests a category for each file — by extension, filename rules, and discovered topics.

STEP 3

edit_note

You review

Edit any category inline, switch between "By type" and "By topic" layouts, and see exactly where each file lands.

STEP 4

drive_file_move

Move or Copy

Apply the plan as a Move or a Copy into an Organized folder. Changed your mind? Undo restores everything.

Two layouts, one scan

Every file is planned two ways at once. Flip between a nested By type tree and a flat By topic view instantly — no re-scan needed.

PRIVACY PROMISE

Your files stay yours

The privacy boundary is built into the code, not just the policy.

lock

Never transmitted

File names and contents never leave your machine. There is no server to send them to.

visibility_off

Contents stay closed

Files are opened only if you opt into "Read file contents" for better categorization.

wifi_off

One opt-in download

The only network call is the optional, one-time AI model download. After that, fully offline.

encrypted

Encrypted logs

Local activity logs are encrypted on disk — not telemetry, and nothing leaves the PC.
