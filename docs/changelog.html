V1.0.0

2026-07-17

STABLE

First public release

FileOrganizerAI 1.0 — a free, fully offline file organizer for Windows, available on the Microsoft Store. Point it at a messy folder, preview a tidy set of category folders, then apply with one-click Undo.

shield PRIVATE BY DESIGN

Your files and their contents are never transmitted. No accounts, no telemetry. The one optional network call is a one-time, opt-in AI model download — decline it and the app never touches the network.

ORGANIZE

One-click, fully offline organization. Pick any folder, hit Analyze, and get a proposed category folder for every file.

Two layouts, one scan. Flip instantly between a nested "By type" tree (Documents/Invoices) and a flat "By topic" view (all Invoices together) — no re-scan.

Preview and edit before anything happens. Review every proposed move and edit any category inline.

Copy with one-click Undo. Files are copied into an Organized folder; your originals stay put, and Undo reverses the last run.

Collision-safe. Duplicate names get a (n) suffix — nothing is ever overwritten.

OPTIONAL ON-DEVICE AI

Group by meaning, not just by name. Turn on "Use semantic AI" and a local neural model suggests smarter categories for tricky files.

Opt-in, one-time download. The model downloads only on your say-so, then runs entirely on your device — no data is sent anywhere.

Fast rules by default. Leave AI off and quick keyword rules do the job, with no download and no network calls at all.

LANGUAGES & PRIVACY

10 languages. English, Spanish, French, German, Portuguese, Italian, Russian, Japanese, Simplified Chinese, and Hindi — auto-detected from Windows and switchable at runtime, no restart.

Contents stay closed. Categorization uses file names and basic properties; files are opened only if you opt into "Read file contents".

Encrypted local activity. On-device activity logs are stored encrypted — not telemetry, and nothing leaves the PC.
